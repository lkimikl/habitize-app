# Habitize — Веб-сервис для мониторинга полезных привычек 💎

**Habitize** — это современное клиентское приложение для формирования и отслеживания прогресса привычек. Проект выполнен в рамках курсовой работы по дисциплине «Фронтенд разработка».

## 🔗 Ссылка на проект
[Посмотреть демо в реальном времени](https://lkimikl.github.io/habitize-app/)

---

## 🚀 Основные функции
* **Трекинг привычек:** Удобная сетка из 28 дней для отслеживания прогресса (в стиле GitHub).
* **Геймификация:** Система уровней (от Новичка до Легенды) и накопление очков опыта (XP).
* **Адаптивный дизайн:** Полная поддержка мобильных устройств.
* **Темная тема:** Автоматическое и ручное переключение цветовой схемы.
* **Интеграция с API:** Ежедневные мотивационные цитаты через внешнее API.
* **Статистика:** Визуализация достижений с использованием графиков (Chart.js).

---

## 🛠 Стек технологий
Согласно техническому заданию, в проекте использованы:
* **HTML5 & CSS3:** Семантическая верстка и адаптивные стили по методологии **БЭМ**.
* **JavaScript (ES6+):** Реализация логики на основе объектно-ориентированного программирования (**ООП**).
* **Vite:** Современный инструмент сборки проекта.
* **Node.js & npm:** Управление зависимостями.
* **Chart.js:** Библиотека для построения графиков статистики.
* **ZenQuotes API:** Внешний сервис для получения контента в реальном времени.
